## 🛠️ Technologies Used
- Node.js
- Express.js
- React
- Css
- Boostrap 5
- Postgre SQL

## 💡Core Concepts
- Rest APIs with Node.js
- Managed database with postgresql
- Single-page pages with React

## 🚀Features
- Customers can
  - Add , update , delete cart order
  - Confirm order 
  
 ## Setup
 ```bash
git clone https://github.com/Metecode/kopernik-hamburger.git
```
frontend:
 ```bash
cd client
```
```bash
npm install
```
```bash
npm start
```
backend:
 ```bash
cd server
```
```bash
npm install
```
```bash
nodemon .\server.js
```
